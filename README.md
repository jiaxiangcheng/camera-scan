# Instalation

1. Install Expo Cli with: `npm install --global expo-cli`
2. `cd camera-scan`
3. `npm install`
4. `npm start`
5. Scan the QR Code with your iPhone to run the app (I dont't have an Android, so I couldn't test it)

-   Dark Mode implemented
